# 🌐 Ewket Forum

> A dynamic, full-stack community Q&A platform designed for interactive knowledge sharing and collaborative problem-solving.

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_Project-blue?style=for-the-badge)](https://ewketforum.netlify.app/)

## 📖 About The Project

**Ewket Forum** is a comprehensive web application where users can ask questions, share insights, and engage in meaningful discussions. Built with a modern tech stack, the platform ensures a seamless user experience, secure backend operations, and a highly responsive interface. 

### 🚀 Live URL
Check out the live application here: **[Ewket Forum Live](https://ewketforum.netlify.app/)**

---

## 🛠️ Built With (Tech Stack)

This project leverages modern web development technologies to ensure performance, scalability, and security:

### Frontend
* **React** - UI Development
* **Vite** - Build Tool & Development Server
* **Axios** - Promise-based HTTP client for seamless API requests
* **Deployment:** Netlify

### Backend
* **Node.js & Express.js** - RESTful API & Server Logic
* **Deployment:** Render

### Database
* **MySQL** - Relational Database Management
* **Hosting:** Aiven Cloud

---

## ⚙️ Features

* **User Authentication:** Secure login and registration systems.
* **Question Management:** Users can post detailed questions and browse existing ones.
* **Interactive Answers:** Community members can contribute answers and solutions.
* **Responsive Design:** Fully optimized for both desktop and mobile devices.
* **Secure API Routing:** Protected backend endpoints with robust CORS configurations.
